# Neon ZigZag — Концепт гри

### 1. Основна дія
Натискання кнопки для миттєвої зміни напрямку руху персонажа по діагоналі з метою уникнення перешкод.

### 2. Керування
Одна кнопка (Пробіл або клік мишкою).

### 3. Мета гри
Протриматися якомога довше в межах коридору, набираючи очки за час виживання.

### 4. Тиск — Обмеження — Наслідок
* **Тиск:** Постійна швидкість руху вгору.
* **Обмеження:** Вузький ігровий простір (червоні стіни).
* **Наслідок:** Зіткнення зі стіною призводить до повної зупинки гри (Game Over).

### 5. Core Loop
Рух вперед -> Клік (зміна вектора) -> Уникнення стіни -> Отримання очок.

### 6. Монетизаційна модель (Ретро-аркада)
* Вартість однієї сесії: 25 центів.
* Мотивація: Побиття власного рекорду ("One more try" loop).
